# Socket-Programming
C code for sending message from Server to Client.
### Running programs
1. Compile
  ```
  gcc tcp_client.c -o client
  gcc tcp_server.c -o server
  ```
2. Run<br>
  First run the server and then the client in seperate terminal windows
  ```
  ./server
  ```
  ```
  ./client
  ```
